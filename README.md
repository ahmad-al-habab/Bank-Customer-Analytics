#  Bank Customer Analytics Dashboard (2022–2024)

##  Author
**Ahmad Al Habab**


##  Project Overview

This project demonstrates an end-to-end Business Intelligence solution for analyzing bank customer and transaction data using SQL Server and Power BI.

The project includes data cleaning, database modeling, advanced SQL analysis, views, stored procedures, and interactive dashboards for business decision-making.

---

## Tools & Technologies

- SQL Server
- T-SQL
- Power BI
- DAX
- GitHub

---

##  Database Structure

The project is built using four main tables:

- Users
- Cards
- Transactions
- MCC_Codes

Relationships were created using Primary Keys and Foreign Keys to build a normalized relational database.

## Power BI Dashboards

The project contains four interactive dashboards:

### Executive Dashboard

- Total Customers
- Total Cards
- Total Transactions
- Revenue Analysis
- Gender Distribution
- Card Brand Analysis

### Customer Analysis

- Customer Segmentation
- Age Distribution
- Income Distribution
- Spending vs Income

### Transaction Analysis

- Revenue by Merchant Category
- Transaction Method Analysis
- Geographic Analysis
- Merchant Performance

### Card & Merchant Analysis

- Card Brand Distribution
- Credit Limit Analysis
- Chip vs Non-Chip Cards
- PIN Change Analysis
